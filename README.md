\# MNIST Digit Classification using CNN



This project implements a Convolutional Neural Network (CNN) for handwritten digit recognition

using the MNIST dataset. The model supports real image upload and prediction.



\## Tech Stack

\- Python

\- TensorFlow / Keras

\- NumPy

\- Matplotlib

\- PIL



\## Features

\- CNN with Batch Normalization and Dropout

\- Data Augmentation

\- Noise Robust Training

\- Custom Image Prediction



\## How to Run

Open `digit\_classifier.ipynb` in Google Colab or Jupyter and run all cells.
